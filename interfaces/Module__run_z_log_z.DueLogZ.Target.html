Interface Target

A message to process before being logged by ZLogger.

Has the same structure as DueLogZ but some properties may be initially omitted. They will be fulfilled by dueLogZ before returned.

Optional index

index: number

Overrides DueLog.Target.index

  • Defined in run-z/log-z/src/message/due-log.ts:94

An index of the first element of the log line to process.

Defaults to 0. When set to negative value, reset to 0. When set to the value greater than the length of the log line, reset to the log line length.

After processing will be set to the value equal or greater then the length of the final log line.
