ZLogLevelFn: { toJSON: any; toLogLevel: any; valueOf: any } & ZLogLevel
  • Defined in run-z/log-z/src/levels.ts:15

A function representing particular log level.

  • Implements a valueOf() method returning a numeric log level value.
  • Implements a toJSON() method returning a numeric log level value.
  • Implements a toLogLevel() method returning a numeric log level value.

zlogExpand

  • zlogExpand(message: ZLogMessage): ZLogMessage
    • Defined in run-z/log-z/src/loggable.ts:116

    Expands a log message by replacing its error and uninterpreted parameters with their loggable representations.

    If error or parameter is a loggable value, then extracts its loggable representation, and processes as following:

    1. ignores null and undefined,
    2. ignores a string equal to the message text,
    3. treats a special value created by one of zlogDetails, zlogError, or zlogExtra accordingly,
    4. processed loggable value recursively,
    5. treats a string as a message text, unless the message has it already, in which case it is treated as uninterpreted parameter,
    6. treats an Error instance as a message error, unless the message has it already, in which case it is treated as uninterpreted parameter,
    7. processes each array element recursively,
    8. treats everything else as uninterpreted parameter.

    Parameters

    • message: ZLogMessage

      A message to expand.

    Returns ZLogMessage

    Expanded log message.
