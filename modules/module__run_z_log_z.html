ZLogLevelFn

ZLogLevelFn: { toJSON: any; toLogLevel: any; valueOf: any } & ZLogLevel
  • Defined in run-z/log-z/src/level/levels.ts:15

A function representing particular log level.

  • Implements a valueOf() method returning a numeric log level value.
  • Implements a toJSON() method returning a numeric log level value.
  • Implements a toLogLevel() method returning a numeric log level value.
  • Implements a toString() method returning a short log level name.
  • When called, returns a log message constructed by zlogMessage with corresponding log level.

zlogDetails

  • zlogDetails(details: ZLogDetails | ((this: void) => ZLogDetails | null | undefined)): ZLoggable
    • Defined in run-z/log-z/src/message/log-details.ts:37

    Creates a loggable value treated as additional message details.

    The resulting value can be passed to zlogMessage function or to any logger method to add details to logged message.

    When processed outside the log-z in the output or default logging stage, the details object is expanded as is, unless the details map is empty. Additionally, if the details value is the last one of the log line, an error detail is extracted from the details map, and logged as the last element of the final log line.

    Parameters

    • details: ZLogDetails | ((this: void) => ZLogDetails | null | undefined)

      Either log message details to add, or a function constructing ones. The function will be called to expand the log message details. It may return null/undefined to expand to nothing.

    Returns ZLoggable

    Loggable value.
